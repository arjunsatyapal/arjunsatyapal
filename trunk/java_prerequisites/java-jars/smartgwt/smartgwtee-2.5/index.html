Starting Points

  • Sample Projects
    Several standalone samples can be found in the distribution under the "samples" directory. Each sample contains a README.txt file that has detailed instruction on building and running the sample.
  • SmartGWT EE Showcase Demo
    A comprehensive SmartGWT Enterprise Edition Showcase demo with source code for each sample viewable by clicking the "View Source" button. The distribution ships with a showcase.war file which you can deploy in your web container to run the Showcase locally.
  • SmartGWT Javadocs
    The SmartGWT Client API documentation in standard Javadoc form.

    The SmartGWT EE Server API documentation in standard Javadoc form.
